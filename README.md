# Brutalist Django TODO App

A simple TODO app with a brutalist design aesthetic, built with Django using Google Antigravity (Gemini 3 Pro).

## Features
- **CRUD Operations**: Manage your tasks.
- **Brutalist UI**: Bold borders, high contrast, and vibrant colors.
- **Tested**: Comprehensive unit tests included.

## Quick Start

1.  **Install dependencies**:
    ```bash
    pip install django
    ```

2.  **Run migrations**:
    ```bash
    python3 manage.py migrate
    ```

3.  **Run the server**:
    ```bash
    python3 manage.py runserver
    ```

4.  **Open**: `http://localhost:8000`

## Running Tests
```bash
python3 manage.py test todo_app
```
